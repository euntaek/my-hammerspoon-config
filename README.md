# my-hammerspoon-config
> `karabiner` 사용하다가 `mx keys`랑 안 맞아서 `hammerspoon`으로 갈아탐.

- **`right option`**
  - `F18` 맵핑 후 한영키로 사용 중
- **`right command`**
  - `vim` 마냥 모드 실행됨 (토글 on/off)
  - `mode on` + `h` 왼쪽 방향키   
  - `mode on` + `l` 오른쪽 방향키   
  - `mode on` + `k` 위 방향키   
  - `mode on` + `j` 아래 방향키   
  - `mode on` + `r` 해머스푼 리로드
